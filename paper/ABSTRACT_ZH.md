# 中文摘要

随机自回归语言生成在不同数值精度之间难以复现，因为早期步骤中的微小分数变化可能改变采样 token，并沿后续前缀持续传播。本文提出稀疏精度重放证书，用公开离散概率合同与目标环境专用修正记录实现精确随机轨迹重放。该方法对相对 logits 进行公开量化，以 token ID 规范候选顺序，将候选权重映射到 (2^{16}) 整数质量空间，并只在目标环境的本地选择不同于参考 token 时记录修正。在 Qwen2.5-1.5B-Instruct 的 20 个中英文 prompts 和 3 个公开 seeds 上，未修正 FP16 到 BF16 重放仅恢复 10/60 条轨迹，使用证书后恢复 60/60 条；平均修正率为 2.16%，prompt-cluster bootstrap 95% 置信区间为 1.80%-2.53%。反向 BF16 到 FP16 实验继续实现 20/20 精确重放。将合同支持从 top-2 扩展到 top-4 降低了截断偏差，但没有降低修正密度，说明分布保真度与跨精度稳定性之间存在 Pareto 权衡。该结论仅适用于已知模型、tokenizer、prompt、公开配置和目标数值环境，不支持跨任意硬件的通用确定性、零分布偏差或语义等价主张。

**关键词：** 随机推理；数值精度；精确重放；语言模型；可复现性；概率合同
