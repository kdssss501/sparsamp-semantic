# R048 双盲文本质量评测方案（草案）

## Material Passport

- Origin Skill: academic-research-suite `experiment-agent`
- Origin Mode: manage
- Ethics Status: `ETHICS_PENDING`
- Data Readiness: `MATERIALS_ONLY`
- Study ID: `R048`

## 研究问题

在相同 Qwen2.5-1.5B-Instruct、系统提示、温度和最大生成长度下，原生 top-16
随机采样、top-2 整数概率合同和 top-4 整数概率合同的文本质量是否存在可观察差异？

本研究只评估公开文本质量，不评估参与者识别生成机制的能力，也不收集敏感信息。

## 设计

- 材料：20 个中英文 prompts，每个 prompt 对应 3 个匿名响应 A/B/C。
- 条件：`native_top16`、`contract_top2`、`contract_top4`。
- 随机化：每个 prompt 的标签排列由 256-bit 私有 key 经 HMAC-SHA256 独立派生。
- 盲法：参与者包不包含方法名称；方法映射只保存在 `outputs/` 下的私有文件。
- 评价维度：相关性、事实/任务正确性、连贯性、语言流畅性，均为 1-5 分。
- 强制选择：每个 prompt 选择总体最佳响应，允许选择“并列/无法判断”。
- 分析单位：参与者内的 prompt；不得把同一参与者的 60 个评分当成独立样本。

## 拟定主要终点

主要终点为总体最佳选择概率。次要终点为四个 Likert 维度的参与者内、prompt 内差异。
分析应使用能够同时处理参与者与 prompt 聚类的模型或分层 bootstrap，并报告效应量与
95% 置信区间；不得只报告逐响应 t 检验或未经校正的多个 p 值。

## 样本量门禁

当前不预设“方便样本”数量。招募前需要：

1. 明确最小实际重要差异；
2. 基于参与者内三条件设计完成保守功效分析；
3. 固定排除规则、主要比较和多重比较校正；
4. 获得学校导师或适用伦理流程的书面确认。

以上门禁完成前，不能把 `ETHICS_PENDING` 改成 `READY`，不能招募或收集评分。

## 纳入与排除（待伦理确认）

- 纳入：年满 18 周岁；能阅读相应语言；自愿同意参加。
- 排除：未完成全部材料；技术故障导致响应未呈现；重复提交。
- 不因评分方向、身份背景或是否符合研究假设而排除。

## 数据最小化

离线页面只导出随机参与者标识、材料签名、评分与完成时间，不收集姓名、邮箱、IP、
设备指纹或自由文本。研究者不得把 blinding key 提供给评分者。揭盲必须在数据冻结与
排除规则执行完成后进行。

## 当前状态

本目录仅包含研究材料草案。`evaluator.html` 在 participant package 为
`ETHICS_PENDING` 时会禁用评分与导出。这一锁定是质量门禁，不代表已经获得伦理批准。
