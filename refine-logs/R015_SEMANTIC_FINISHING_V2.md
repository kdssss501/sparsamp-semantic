# R015 认证语义收尾 V2

日期：2026-07-17

## 改动

V2 在句末检测前排除独立行上的数字列表标记，例如 `1.` 和 `12.`。采样器、payload、
SparSamp block 和最大尾部预算均未改变。

- 平均尾部 17 tokens，平均 finishing latency 0.993 秒，平均 visible bit/token 0.992。
- Prompt 1 不再停在列表编号 `1.`，而是生成完整条目“确保AI内容的准确性和多样性，避免
  偏见和错误。”

## 解释

V2 证明语义收益来自公开停止规则，而不是修改隐写采样。相比固定 16/32-token 尾部，结构
感知标点模式能用可变长度在完整句末停止；代价是平均现实传输容量低于立即停止。当前只有
3 个 prompt，下一阶段需要 20 prompts x 3 seeds，并加入人工盲评或自动句法完整性标签。
